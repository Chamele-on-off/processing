Документация API
Базовый URL:
https://ваш-домен/api/v1/
Пример запроса:
POST /orders/create
Headers:
- X-API-KEY: ваш_api_ключ
- X-API-SECRET: ваш_секретный_ключ
Content-Type: application/json

Body:
{
    "amount": 1000,
    "currency": "RUB",
    "type": "deposit",
    "requisites": {
        "type": "bank",
        "bank_name": "Сбербанк",
        "bik": "044525225",
        "account_number": "40817810099910004312",
        "account_holder": "Иванов Иван Иванович"
    }
}
Доступные методы:
  • POST /orders/create - Создать новую транзакцию
  • GET /orders/{id} - Получить информацию о транзакции
  • GET /balance - Получить текущий баланс
  • GET /transactions - Получить список транзакций
  • POST /transactions/{id}/cancel - Отменить транзакцию
